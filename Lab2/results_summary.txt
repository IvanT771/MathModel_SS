Практическое занятие №2. Уравнение параболического типа. Явная схема.

Параметры базовой модели: L = 1.0, n = 5, dx = 0.20000, λ = 0.13333, ρ = 0.10000, c = 1.00000, a = 1.33333.

Задание 1. Верификация модели (ГУ 1-го рода, аналитическое решение).
Параметры модели: dx = 0.20000, a = 1.33333.
Показатель устойчивости r = 0.22222.
Максимальная ошибка при t = 4 c: 5.684342e-14.
Максимальная ошибка при t = 10 c: 5.684342e-14.
Максимальная ошибка при t = 20 c: 5.684342e-14.
Максимальная ошибка при t = 50 c: 5.684342e-14.
Отклонение от стационарного линейного решения при t = 50 c: 5.684342e-14.

Задание 2. Проверка устойчивости явной схемы.
Базовый случай: r = 0.22222 <= 0.5, схема устойчива.
При увеличении λ в 10 раз: r = 2.22222 > 0.5, схема неустойчива.
Максимальная по модулю температура в неустойчивом режиме к t ≈ 0.033 c: 7.698e+04.

Задание 3. Исследование влияния временного шага при ГУ 1-го рода.
Для m = 300: dt = 0.00667, r = 0.22222.
Для m = 600: dt = 0.00333, r = 0.11111.
Максимальное расхождение между m=300 и m=600 при t ≈ 0.027 c: 1.740391.
Максимальное расхождение между m=300 и m=600 при t ≈ 0.067 c: 0.698727.
Максимальное расхождение между m=300 и m=600 при t ≈ 0.133 c: 0.449211.
Максимальное расхождение между m=300 и m=600 при t ≈ 0.333 c: 0.078865.

Задание 4. Исследование процесса при ГУ 3-го рода.
T1=150, α1=1; T2=40, α2=1: r = 0.22222, q = 11.57895, u_left(2 c) = 138.421, u_right(2 c) = 51.579, аналитика = (138.421; 51.579).
T1=150, α1=0.1; T2=40, α2=1: r = 0.22222, q = 5.94595, u_left(2 c) = 90.538, u_right(2 c) = 45.945, аналитика = (90.541; 45.946).
T1=150, α1=1; T2=40, α2=0.1: r = 0.22222, q = 5.94595, u_left(2 c) = 144.052, u_right(2 c) = 99.453, аналитика = (144.054; 99.459).
T1=150, α1=0.1; T2=40, α2=0.1: r = 0.22222, q = 4.00000, u_left(2 c) = 109.322, u_right(2 c) = 79.322, аналитика = (110.000; 80.000).
Стационарные значения для случаев α1=α2=1, α1=0.1/α2=1, α1=1/α2=0.1 и α1=α2=0.1 согласуются с формулами тепловых сопротивлений.
Для случая α1=α2=0.1 к t = 2 c решение еще не полностью достигает стационарного режима, что видно по малому, но ненулевому отличию от аналитического предела.

Выводы.
1. Явная схема корректно воспроизводит аналитическое решение при выполнении критерия устойчивости.
2. Условие r <= 0.5 является определяющим: при увеличении коэффициента теплопроводности в 10 раз схема расходится.
3. Уменьшение временного шага с m=300 до m=600 меняет решение слабо, то есть модель демонстрирует сходимость.
4. При ГУ 3-го рода стационарные температуры на границах зависят от дополнительных тепловых сопротивлений 1/α.
5. Чем меньше коэффициент теплоотдачи α, тем сильнее температура на соответствующей границе отходит от температуры внешней среды.
